# OverseerParser
Parses EverQuest Overseer Data

Requires:
dbstr_us.txt
OvrJobClient.txt
OvrMiniClient.txt
OvrMiniTraitClient.txt
OvrQstClient.txt

Parses these out to:
agentnames.txt:
Contains the Agents in ID | Name | Tier | Iconic | Convert to Retire? | Job + Level | Traits

OverseerQuests.txt
Contains the Quests in ID | Name | Tier | Type | ~Base % Success
